Plongez dans l'Ombre

Un jeu d'infiltration en 2D où chaque mouvement compte

Exploration

Parcourez des niveaux labyrinthiques plongés dans l'obscurité. Chaque recoin cache un danger… ou un trésor.

Discrétion

Les gardes patrouillent avec des cônes de vision. Restez hors de leur champ de lumière ou c'est la fin.

Collecte

Ramassez émeraudes et diamants disséminés dans les niveaux. Plus vous en collectez, meilleur est votre score.

Sous le Capot

Développé entièrement en Python avec Pygame, Shadow Thief repose sur une architecture client-serveur pensée pour le multijoueur.

Python Pygame Sockets Client-Serveur
  • Moteur de jeu personnalisé avec gestion de collisions
  • Système de lumière et cônes de vision dynamiques
  • Mode multijoueur en temps réel
  • IA de patrouille pour les gardes
  • Niveaux thématiques : Labyrinthe, Forteresse, Entrepôt
